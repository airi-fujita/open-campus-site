# open-campus-site

公開URL：
https://airi-fujita.github.io/open-campus-site/

授業課題として制作した架空の専門学校のオープンキャンパス特設サイトです。
調理・製菓・栄養について学べる専門学校のオープンキャンパス特設サイトを制作しました。

高校生が気軽に参加したいと思える親しみやすさと、安心して進学先をして検討できる信頼感の両立をコンセプトとして制作しました。

デザイン面では、丸みのあるフォント、角丸のカード、ベースカラーは暖色のオレンジを使用し、親しみやすい印象を表現しています。一方で、テキストカラーや背景色にグラデーションを取り入れることで、フォーマルな印象となり、学校としての信頼感を出せるようにしました。

また、ナビゲーションバーを設置し、必要な情報へすぐにアクセスできる構成にしています。
さらに、カードにアニメーションをつけたり、画像のスライドショーを取り入れ、学校の魅力を視覚的に伝えながら、最後まで飽きずに閲覧できる工夫を行いました。
申込ボダンはナビゲーションバー、開催日時直下、ページ下部に配置し、参加を検討したタイミングでスムーズに申し込める導線設計を意識しました。
また、よくあるご質問ではアコーディオン形式を採用し、必要な情報だけを効率よく確認できるようにすることで、視認性と操作性の向上を図りました。
